# ComPASS 项目记忆

最后更新：2026-06-18

## 产品定位

ComPASS 我的求职指南针，是面向在日本找工作的留学生的求职辅助网页。目标用户包括本科、大学院、语言学校毕业或即将毕业的学生。

核心目标：
- 按条件筛选正在招聘、开放实习、适合外国人/留学生的日本公司。
- 展示公司日本名称、业务内容、外国人採用比率、口コミ评分、签证支援、応募截止等信息。
- 帮用户管理求职进度、資格学习、ES 改进、面试准备、个人文件和日程。
- 网页先做成熟，后续再考虑 iOS App，并与网页数据互通。

## 数据真实性原则

公司数据、活动数据、资格日程都可能变化。

实现时要区分：
- 官方公开
- 平台公开
- 候选数据
- 示例数据
- 待确认
- 未公开

不要把示例数据写成已爬取真实数据。
需要真实最新活动/招聘/报名日期时，必须联网查官方页面。
